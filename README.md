## CHAT SERVICE

#### Technologies

React + Electron + Socket.io

#### How to use

- Start server

```
$ cd web/server
$ yarn install && yarn start
```

- Start React app

```
$ cd web/client
$ yarn install && yarn start
```

- Start electron

```
$ cd native
$ yarn install && yarn start
```
